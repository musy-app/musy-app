Kitchen is a [styled-components](https://styled-components.com) suite of components that are designed to work together to create a beautiful and consistent UI. Inspired by [Geist](https://geist-ui.dev/) and [Vercel Design](https://vercel.com/design) with a focus on simplicity and ease of use. This library is maintained by [Tonight Pass](https://tonightpass.com).

<Callout>
  This package is still in active development and is not yet ready for
  production use.
</Callout>

## Installation

<Tabs items={["npm", "yarn"]}>
  <Tab>
    Kitchen is available as an [npm package](https://www.npmjs.com/package/@tonightpass/kitchen).

    ```bash
    npm i @tonightpass/kitchen --save
    ```

  </Tab>
  <Tab>
    Kitchen is available as a [yarn package](https://yarnpkg.com/package/@tonightpass/kitchen).

    ```bash
    yarn add @tonightpass/kitchen
    ```

  </Tab>
</Tabs>

After installing Kitchen, you need to set up the `KitchenProvider` at the root of your application.

```jsx
import { KitchenProvider } from "@tonightpass/kitchen";

function App({ Component, pageProps }) {
  return (
    <KitchenProvider>
      <Component {...pageProps} />
    </KitchenProvider>
  );
}

export default App;
```
